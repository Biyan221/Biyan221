Garansi jika barang diterima dan terjadi masalah saat pemakaian awal bisa di garansi.

Cek adaptor powersuply, beberapa type charger kurang bersih recomended pakai adaptor efek stombox 9 v negativ center, pilihan lain powerbank ketiga charger tipe ori, atau bila rekam di stage melalui mixer sebaiknya memakai pasif mode (kplug tidak di suply agar perbedaan grounding dari mixer dan device bisa di minimalkan.

Cek sinyal input yang masuk ke hp perekam, untuk hp yang plug n play (samsung iphone sony) bisa cek grafik audio input di aplikasi perekam audio bawaan. Beberapa tipe dan masing masing merk Hp mungkin bisa berbeda sensitivitas hasil rekamnya, jika kurang bisa di mastering volume saat potong atau edit video dengan aplikasi seperti kinemaster. Setting input Kplug 1c setara musik mixing, untuk beberap alat musik pasif mungkin harus menggunakan efek pedal atau preamp external.

Untuk Kplug tipe 1c fitur 2 pasif mono input tidak memungkinkan menggunakan mic dinamic maupun kondenser secara direct langsung. Untuk menggunakan mic harus melalui mixer atau efek vokal terlebih dahulu sebelum masuk kplug tipe 1c.5. Untuk type V.1D sudah support mic input.

Untuk video record pada android memakai aplikasi "K-plug cam studio" atau aplikasi pihak ke tiga video atau audio record yang support sumber external audio.

Untuk Ios ( Iphone ) Sudah support pada aplikasi video atau audio record bawaan menggunakan adapter lightning to 3.5 mm yang original atau yang bersertifikat MFI ( Made for Iphone)

Cek kabel Routing Input. Untuk kabel yang disarankan menggunakan input kabel mono TS 6,5 mm, untuk k-plug type V.10 Cek Switch "Input A" Saat Off Adalah mode line input, saat on mode microphone dynamic input.

K-plug V.1C / V.1D Input mode line in, untuk instument pasif ( bass pasif / gitar pasif pick up ) harus menggunakan pre amp external ataupun efek gitar.

Untuk K-plug V.1C hanya terdapat fitur 2 line Input mode. Untuk K-plug V.1D selain 2 line input, input A sudah support microphone input ( dynamic 6.5 mm atau condenser 3.5 mm ). V.1D juga terdapat switch mode rack untuk menambahkan aplikasi efek pada smartphone ataupun tracking audio di apps DAW smartphone, maupun live streamming atau komunikasi 2 arah.
